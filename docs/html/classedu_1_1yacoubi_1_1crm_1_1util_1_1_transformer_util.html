Detailed Description

Utility-Klasse für allgemeine Transformationsmethoden.

Diese Klasse stellt eine universelle Methode zur Verfügung, um beliebige Transformationen mithilfe des ITransformer Functional Interface durchzuführen. Sie ermöglicht es, eingehende Entitäten in verschiedene Zieltypen zu transformieren, ohne die Logik der Transformationsmethode zu ändern.

Beispielverwendung:



Transformer<SourceType, TargetType> transformer = source -> new TargetType(source.getProperty());

TargetType target = TransformerUtil.transform(transformer, sourceInstance);

Die Klasse kann nicht instanziiert werden, da der Konstruktor privat ist.

T entity )
inlinestatic

Universelle Transformationsmethode, die den übergebenen ITransformer auf die Entität anwendet.

Diese Methode nimmt eine Entität und einen ITransformer entgegen und gibt die transformierte Entität zurück. Sie ist nützlich für allgemeine Transformationen zwischen verschiedenen Typen.

Beispiel:



Transformer<Note, NoteDTO> noteToDtoTransformer = note -> new NoteDTO(note.getId(), note.getContent());

NoteDTO noteDto = TransformerUtil.transform(noteToDtoTransformer, noteInstance);
