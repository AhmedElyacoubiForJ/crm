R transform (T t)
 

Detailed Description

Functional Interface für die Transformation von Entitäten und DTOs.

Dieses Interface definiert eine Methode zur Transformation einer Entität vom Typ T in einen Zieltyp R. Es kann verwendet werden, um allgemeine Transformationen zwischen verschiedenen Typen zu abstrahieren und wiederverwendbare TransformationsLogik zu schaffen.

Beispielverwendung:



Transformer<Note, NoteDTO> noteToDtoTransformer = note -> new NoteDTO(note.getId(), note.getContent());

NoteDTO noteDto = noteToDtoTransformer.transform(noteInstance);
Parameters
<T>der Eingabetyp der zu transformierenden Entität
<R>der Rückgabetyp der transformierten Entität
